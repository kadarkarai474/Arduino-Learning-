Arduino LED Blinking (2 Seconds)
Description

This project demonstrates basic LED blinking using Arduino Uno in the WOKWI ONLINE SIMULATION. The LED connected to pin 13 turns ON for every 2 seconds and remains OFF for TWO seconds.
Components Used

    Arduino Uno , LED , 220k Resistor ,connecting wires

Working

The LED blinks continuously: ->ON for 2 seconds ->OFF for 2 seconds
Code

Located in: 01_LED_Blink/sketch.ino
Learning Outcome

-> Understanding digital output using DigitalWrite() -> Using delay() function -> Basic Arduino programming
